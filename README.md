Catálogo de Filmes e Séries
Este é um projeto de estudo desenvolvido para consolidar conhecimentos em HTML e CSS. O site consiste em uma vitrine de recomendações de filmes e séries com um design moderno, sombrio e responsivo.

Sobre o Projeto
O objetivo deste projeto foi criar uma interface de catálogo (estilo streaming) utilizando apenas tecnologias web fundamentais. O projeto foca em:

Estruturação semântica de documentos HTML.

Estilização avançada com CSS (Flexbox, Hover Effects, Box Shadows).

Criação de animações para interatividade do usuário.

Tecnologias Utilizadas
HTML5: Para a estruturação do conteúdo e acessibilidade.

CSS3: Para o layout, design visual e animações.

Google Fonts / Font-family: Utilização de fontes modernas para melhor legibilidade.

Funcionalidades
Header Sticky: Menu de navegação que acompanha a rolagem da página.

Grid de Conteúdo: Exibição organizada de cards de filmes e séries.

Efeitos de Hover: Os cards aumentam de escala e ganham destaque ao passar o mouse.

Design Dark Mode: Interface otimizada para visualização confortável em ambientes escuros.

Seção de Contato: Informações para feedback e sugestões.

Como Executar
Não é necessário instalar nenhuma dependência ou servidor para visualizar o projeto.

Faça o download ou clone este repositório:

Bash
